# Gemgrapher

CLI tool to generate gem dependency graph from your gemfile

## Installation

```
gem install gemgrapher
```

## Usage

Go to the root dir of your ruby app and run 
``` 
gemgrapher
```
This will generate the html in place. Open index.html generated to show an interactive Hierarchical Edge Bundling graph.
